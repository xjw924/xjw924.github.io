WOE与IV

理论

WOE定义

全称：Weight of Evidence

前提：计算之前需要是离散化后的连续变量or离散变量。

对于变量第$i$个取值的$WOE_i$的计算公式为

其中$ # y_i $表示在第$i$个取值的样本中（或第$i$个箱内）的正样本个数，$ # y_T $表示所有正样本的个数，$py_i$表示在第$i$个取值的样本中（或第$i$个箱内）正样本占所有正样本的比例。

从最后一个等号后的表达式可以看出，$WOE_i$表示的是某个取值下正样本和负样本的比值，与所有样本中这个比值的差异。这个差异是用比值再取对数来表示的。$WOE_i$越大，表示差异越大，则这个取值下的样本为正的可能性越大。

$WOE_i$编码作用

  1. 标准化功能。编码过后的自变量其实具备了某种标准化的性质，自变量内部的各个取值之间都可以直接进行比较，且不同自变量之间的各个取值也可以通过$WOE_i$进行直接的比较。
  2. 可以反映出自变量的贡献情况（？）。自变量内部$WOE_i$值的变异（波动）情况，结合模型拟合出的系数，构造出各个自变量的贡献率和相对重要性。一般地，系数越大，$WOE_i$的方差越大，则自变量的贡献率越大。

IV​

全程：Information Value

在$WOE_i$的基础上，$IV$在$WOE_i$的前面乘以一个系数$(py_i-pn_i)$作为各箱$WOE_i$的权重，并进行加权求和，其计算公式如下

其中$N$为分箱的个数（变量的取值个数），系数$(py_i-pn_i)$为箱内正样本占比与负样本占比的差。该系数不仅保证了每个箱乘积的值为非负数，更重要的是考虑了变量当前箱中样本的数量占整体样本数量的比例，比例越高，该箱对变量整体预测能力的贡献越高。

$IV$衡量某个特征对目标的影响程度，通过该特征中正负样本的比例与总体正负样本的比例，来对比和计算其关联程度，因此可以代表该特征上的信息量以及该特征对模型的贡献。

$IV$是对于整个特征来说的，代表的意义由下表来控制：

IV 特征对预测函数的贡献
<0.03 特征几乎不含有效信息，对模型没有贡献，可以删除
[0.03, 0.10) 有效信息很少，对模型的贡献度低
[0.10, 0.30) 有效信息一般，对模型的贡献度中等
[0.30, 0.50) 有效信息较多，对模型的贡献度较高
>=0.50 有效信息非常多，对模型的贡献极高且可疑

因此通常会选择$IV$值在0.1~0.5范围内的特征。

单机python

def CalcWOE(df, col, target):
    '''
    获得某个变量的分箱操作后每个箱体对应的WOE值，并且基于WOE值计算该变量的IV值
    :param df: 包含需要计算WOE的变量和目标变量
    :param col: 需要计算WOE、IV的变量，必须是分箱后的变量，或者不需要分箱的离散型变量
    :param target: 目标变量，0、1表示好、坏
    :return: 返回WOE和IV
    '''
    total = df.groupby([col])[target].count()  # 每个箱体的总数
    total = pd.DataFrame({'total': total})    
    bad = df.groupby([col])[target].sum()      # 每个箱体的坏样本数
    bad = pd.DataFrame({'bad': bad})
    regroup = total.merge(bad, left_index=True, right_index=True, how='left')
    regroup.reset_index(level=0, inplace=True)
    N = sum(regroup['total']) # 总体样本数
    B = sum(regroup['bad'])   # 总体坏样本数
    regroup['good'] = regroup['total'] - regroup['bad']  # 每个箱体的好样本数
    G = N - B                 # 总体好样本数
    regroup['bad_pcnt'] = regroup['bad'].map(lambda x: x/B)    # 每个箱体的坏样本数占总体的坏样本数的比例
    regroup['good_pcnt'] = regroup['good'].map(lambda x: x/G)  # 每个箱体的好样本数占总体的好样本数的比例
    # WOEi计算公式(不含系数)： WOE(每个箱体) = ln(该箱体的好样本数占总体的好样本数的比例/该箱体的好样本数占总体的好样本数的比例）
    regroup['WOE'] = regroup.apply(lambda x: np.log(x.good_pcnt/x.bad_pcnt),axis=1)   
    WOE_dict = regroup[[col,'WOE']].set_index(col).to_dict(orient='index')
    for k, v in WOE_dict.items(): # k代表箱体名，v代表了以WOE为键，箱体的实际WOE值为value的字典
        WOE_dict[k] = v['WOE']
    # 计算该变量的IV值：sum((某箱体的好样本数占总体的好样本数的比例 - 该箱体的坏样本数占总体的坏样本数的比例)*WOE(某个箱体))
    IV = regroup.apply(lambda x: (x.good_pcnt-x.bad_pcnt)*np.log(x.good_pcnt/x.bad_pcnt),axis = 1)
    IV = sum(IV)
    return {col: {"WOE": WOE_dict, 'IV':IV}}

集群pyspark

def cal_woe_iv(tableName, featureCol, labelCol):
     '''
     计算spark.sql.DataFrame的WOE和IV
     :param tableName: 储存在HDFS上的表名
     :param featureCol: 所需要计算的字段名
     :param labelCol: 标签列名
     :return WOE字典和IV值
     '''
     df = spark.sql("select {featureCol}, {labelCol} from {tableName}".format(featureCol=featureCol, labelCol=labelCol, tableName=tableName))
     # 使用crosstab函数生成列联表，并对列进行命名
     binCount = df.crosstab(featureCol,labelCol).toDF('feature','neg_cnt','pos_cnt')
     # 注册临时表便于后续调用
     binCount.registerTempTable("binCount")
     # 计算主要步骤
     woe_df = spark.sql("""
        select
          feature,
          nvl(log(pos_per/neg_per), 0) as woe_i,
          nvl((pos_per-neg_per)*log(pos_per/neg_per), 0) as iv_i
        from
        (
          select
            feature,
            neg_cnt/(select sum(neg_cnt) from binCount) as neg_per,
            pos_cnt/(select sum(pos_cnt) from binCount) as pos_per
          from
            binCount
        ) t
     """)
     # 将计算结果转化为pandas.DataFrame
     woe_df_pandas = woe_df.toPandas()
     woe = dict(zip(woe_df_pandas["feature"].values.reshape(-1,), woe_df_pandas["woe_i"].values.reshape(-1,),))
     iv = sum(woe_df_pandas["iv_i"].values)
     return woe, iv

if __name__ == '__main__':
    from pyspark.sql import SparkSession
    import time
    import numpy as np
    spark = SparkSession.builder.enableHiveSupport().getOrCreate()
    tableName = "test.calc_woe_iv_sample"
    featureCol = "age_level"
    labelCol = "label"
    time_cost = []
    for i in range(30):
        start = time.time()
        woe, iv = cal_woe_iv(tableName, featureCol, labelCol)
        end = time.time()
        time_cost.append(end-start)
    print ('time cost: %.5f sec' % (np.mean(time_cost)))
    print ('woe:', woe)
    print ('iv:', iv)
